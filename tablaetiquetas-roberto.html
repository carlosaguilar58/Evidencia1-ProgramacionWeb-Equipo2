Etiquetas utilizadas en mi CV y Portafolio (versión minimalista)

Esta página resume las etiquetas HTML y las ideas de CSS que aplicamos en tu currículum y portafolio minimalistas. Incluye ejemplos, bloques de código y una plantilla lista para copiar.

Estructura básica del documento

Define el esqueleto del HTML para idioma, codificación y título.

<!DOCTYPE html>
<html lang="es">
  <head>
    <meta charset="UTF-8">
    <title>Mi CV / Portafolio</title>
  </head>
  <body>
    <!-- Contenido visible -->
  </body>
</html>

Etiquetas semánticas (layout)

Organizan el contenido y mejoran SEO/accesibilidad.

  • <header> Encabezado (nombre, título, contactos).
  • <main> Contenido principal (perfil, proyectos).
  • <aside> Información lateral (habilidades, idiomas).
  • <section> Bloques temáticos dentro de main/aside.
  • <footer> Pie de página.
<header>Encabezado con nombre y datos</header>
<main>Perfil, educación, proyectos</main>
<aside>Habilidades e idiomas</aside>
<footer>© Roberto</footer>

Texto y jerarquía (h1–h6, p, strong, em, span)

Los encabezados establecen jerarquía; los párrafos y estilos dan claridad.

  • <h1>…<h6> Títulos de mayor a menor importancia.
  • <p> Párrafo.
  • <strong> Énfasis importante (lectores lo resaltan).
  • <em> Énfasis leve.
  • <span> Contenedor en línea (útil para chips/badges).

Ejemplos rápidos:

<h1>Roberto A. Álvarez Sandoval</h1>
<h3>Estudiante de Tecnologías de la Información</h3>
<p>Perfil profesional…</p>
<p>Habilidades: <strong>Python</strong>, <em>aprendizaje continuo</em></p>

Listas (ul, ol, li)

Resumir habilidades, logros o puntos clave.

<ul>
  <li>Python</li>
  <li>SQL</li>
  <li>Análisis de datos</li>
</ul>

Enlaces (a, mailto, tel)

Para correo, teléfono o sitios externos (GitHub, LinkedIn).

<a href="mailto:8118470080r@gmail.com">Enviar correo</a>
<a href="tel:8127004070">Llamar</a>

Imágenes (img, alt, tamaños)

Foto con accesibilidad y control visual (recorte circular con CSS).

<img src="foto.jpg" alt="Foto de Roberto Angel Álvarez Sandoval" width="160">

Usamos object-fit: cover; y border-radius: 50%; para el estilo circular.

Clases y atributos comunes (class, id, lang, charset)

Para identificar/estilizar elementos con CSS y definir idioma/codificación.

  • class: estilos reutilizables (.photo, .badge).
  • id: identificador único (anclas del menú).
  • lang: idioma del documento (lang="es").
  • charset: codificación (UTF-8).
<h1 id="perfil">Perfil</h1>
<span class="badge">Python</span>

CSS clave del diseño minimalista

Variables, tipografía, colores, bordes y espaciado.

:root {
  --bg:#ffffff; --fg:#0f172a; --muted:#6b7280; --line:#e5e7eb;
}
body { background:var(--bg); color:var(--fg); font:16px/1.65 system-ui,Arial,sans-serif; }
.badge { border:1px solid var(--line); border-radius:999px; padding:6px 10px; }
.photo img { width:100%; height:100%; object-fit:cover; border-radius:50%; }

Apostamos por alto contraste, tipografía legible y “aire” entre bloques.

Grid y diseño responsivo (@media)

Para que se vea bien en escritorio y móvil.

.wrapper {
  display:grid;
  grid-template-columns:320px 1fr;
}
@media (max-width: 860px) {
  .wrapper { grid-template-columns:1fr; }
}

Badges de habilidades e idiomas (span + class)

Chips limpios para listar habilidades e idiomas.

Python SQL Análisis de datos Español (nativo) Inglés (básico)

Accesibilidad básica (alt, aria-label)

Buenas prácticas aplicadas:

  • alt descriptivo en imágenes.
  • Contraste suficiente de color.
  • Jerarquía correcta de encabezados.
  • aria-label cuando el texto visible no sea claro (opcional).
<a href="mailto:..." aria-label="Enviar correo a Roberto">Correo</a>

Plantilla base reutilizable

Copia/pega para empezar un nuevo bloque.

<section class="card">
  <h2>Título de sección</h2>
  <ul>
    <li>Punto clave 1</li>
    <li>Punto clave 2</li>
  </ul>
</section>
